Kings County Sales Dashboard

House sales analysis in Kings County, Washington using Tableau.

Project Summary

This project entails analyzing house sales data in Kings County, Washington, using Tableau. The dashboard visualizes average house prices, sales distributions, and other key metrics to provide insights into the real estate market.

Process

1. Data Collection: Sourced the data from Kaggle.
2. Data Cleaning: Performed initial data cleaning and transformation in Tableau.
3. Data Analysis: Created calculated fields, parameters, and measures to analyze the data.
4. Data Visualization: Developed various charts, including a line chart for average prices, a map for prices by zip code, distribution charts, and bar charts for bedrooms and bathrooms.

Challenges and Solutions

One challenge was handling missing values and outliers, which were addressed by thorough data cleaning and validation. Another challenge was designing an intuitive and interactive dashboard, achieved by implementing filters and calculated fields effectively.

Outcome and Impact

The project provided valuable insights into the Kings County real estate market, helping stakeholders make informed decisions about buying, selling, and investing in properties. Future improvements include incorporating more data sources and advanced analytics.
