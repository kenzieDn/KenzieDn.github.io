London Bike Rides

Data extraction and visualization using Python and Tableau.

Project Summary

This project involves extracting and visualizing data on London bike rides using Python and Tableau. Data was sourced from Kaggle, extracted using Python, and visualized in Tableau to uncover patterns and insights about bike ride trends in London.

Process

1. Data Extraction: Using Kaggle's API and Python libraries to download and extract the data.
2. Data Cleaning: Processed the data in Python to clean and transform it, such as renaming columns and converting integer values to strings.
3. Data Analysis: Created parameters and calculated fields in Tableau to analyze the data.
4. Data Visualization: Developed various charts, including a moving average line chart, heatmap, and bar charts to visualize bike ride trends, weather conditions, and more.

Challenges and Solutions

A key challenge was accurately mapping integer values to their respective string representations, which was solved using dictionaries in Python. Additionally, ensuring the visualizations in Tableau were interactive and insightful required creating dynamic reference bands and callout charts.

Outcome and Impact

The project provided a comprehensive analysis of London bike ride patterns, highlighting the impact of weather and time on ride frequency. This information is valuable for city planners and bike-sharing companies to optimize their services and infrastructure.
