Excel Coffee Sales

Data visualization and analysis on coffee sales.

Project Summary

This project involves using Excel to visualize and analyze coffee sales data. The goal is to extract meaningful insights to help improve sales strategies and customer targeting. Tools used include Excel for data cleaning, manipulation, and visualization. The dashboard features a timeline for selecting a time period, two bar charts, three slicers, and one main line chart.

Process

1. Data Collection: The data was collected from various sources, including sales records, customer feedback, and market research reports.
2. Data Cleaning: The data was cleaned to remove any inconsistencies, duplicates, and errors. I utilized the XLOOKUP formula to match new columns created in the main sheet with data in other sheets.
3. Data Analysis: Various Excel functions and formulas were used to analyze the data, including pivot tables, XLOOKUP, TRIM and data validation.
4. Data Visualization: Charts and graphs were created to visualize the trends and patterns in the data. The visualization features a timeline to select a time period, two bar charts, three slicers, and one main line chart. The slicers include size, roast type, and loyalty card, while the timeline uses the order date. The visualizations show total sales over time in the line chart, sales by country, and the top 5 customers in bar charts.

Challenges and Solutions

One major challenge was handling missing data. This was addressed by using Excel's data validation and conditional formatting features to identify and fill in the gaps. Another challenge was ensuring the accuracy of the data analysis, which was mitigated by cross-verifying the results with multiple Excel functions. The XLOOKUP formula was extensively used to match new columns in the main sheet with other sheets. A minor challenge was ensuring the dashboard remained user-friendly despite the complexity of the data. Achieved by using clear and interactive visual elements.

Outcome and Impact

The project successfully identified key sales trends and customer preferences, leading to a more targeted marketing strategy and increased sales. Future improvements include automating data collection and integrating the analysis with other business intelligence tools.
